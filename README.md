# CustomKick (Executor)

## Usage

### Load

    local KickPlayer = loadstring(game:HttpGet("https://raw.githubusercontent.com/TheNexusZen/CustomKick/refs/heads/main/Kick.lua"))()

---

### Basic

    KickPlayer("<C=r>BANNED</c>", "<C=w>You have been permanently banned.</c>")

---

## Parameters

| Parameter | Type   |
|----------|--------|
| title    | string |
| reason   | string |

---
